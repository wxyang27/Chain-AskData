# schema_indexing

Offline schema and knowledge indexing layer.

| Capability | Module |
|---|---|
| Raw knowledge import | `app.knowledge_importer` |
| Schema index build | `app.schema_indexing.builder` |
| Schema index load | `app.schema_indexing.loader` |
| Chroma/vector store | `app.knowledge_indexer` |
| Offline build command | `app.schema_indexing.build_indexes` |
